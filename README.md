# POSL 物品管理自動化ツール

## 開発背景

現在 POSL では，物品管理係は私含め 2 人いるが，管理が追いついていないのが現状である．物品管理の属人化を解消するため，Slack App による物品管理自動化ツールを作成する．

## 機能要件

- Slack から物品を登録できる
- Slack から物品を貸し出しできる
- Slack から物品を返却できる
- Slack から物品を削除できる

## 追加要件

- Slack から物品の発注依頼ができる
